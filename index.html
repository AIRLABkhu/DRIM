DRIM Dataset

DRIM : Depth Restoration with Interference Mitigation
in Multiple LiDAR Depth Cameras

Kyung Hee University AIRLAB
Paper Github Dataset

Dataset Structure

  • This dataset provides Interference Affected Depth Data and Ground Truths from Multiple Intel Realsense L515 Systems.



Data

  • Raw Data (640 x 480)
    • Interference Unaffected Depth (1 channel, uint16)
    • Interference Unaffected RGB (3 channel, uint8)
    • Interference Affected Depth (1 channel, uint16)
    • Interference Affected RGB (3 channel, uint8)

  • Ground Truth (640 x 480)
    • Interference Artifact Mask (1 channel, uint8)
    • Sensor Artifact Mask (1 channel, uint8)
    • GT Depth Data (1 channel, uint16)

Categories (Directory Name)

  • We collected a dataset from various objects classified into four different categories.
    • Grocery (13 objects)
    • Kitchen (15 objects)
    • Office (18 objects)
    • Toy (17 objects)

Data Directory

  • Our dataset includes two kinds of data. Raw_dataset is using for training directly. Our model was trained with this format.
  • Classify_dataset is classfied for each objects. Users can select useful object data. Our dataset directory structure is below.
